CRM Masjid Lite

Merupakan software berbasis website dan android. Projek ini dilakukan pada saat Internship di PT. Bejana Investidata Globalindo. Dimana saya bertugas untuk membuat API dan mentestingnya. Projek ini dibuat dengan menggunakan framework PHP yaitu Laravel. Beberapa fungsi yang dibuat antara lain fungsi login, registrasi, verifikasi email, CRUD data, dan lainnya.
